  - components 
    - Elements => 재사용 가능한 input button 등을 모아 놓는 폴더 
    - Layout => Home / workspace에서 사용하는 사이드바 헤더를 모아 놓는 폴더 
  
  - pages
    - page들을 모아 놓는 폴더
  
  - shared
    - Atoms  => recoil의 atom을 모아놓는 폴더 
    - Cookie => Cookie 관련 코드 
    - Router => Router 관련 코드 
    - Styles => GlobalStyle / custom style을 모아놓는 폴더 

  - utils
    - util 함수들을 모아 놓는 폴더
